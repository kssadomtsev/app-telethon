Список доступных команд:

  1. list - отображает список отслеживаемых каналов
  2. add <cсылка на канал> - добавление нового канала в список отслеживаемых. Пример: add https://t.me/Krololochannel или: add @grustnie_memi
  3. delete <ID канала> - удаляет канал из списка отслеживаемых по его ID (ID видно в выводе команды list). Удаляются все посты из этого канала из базы данных и бот покидает этот канал
  4. dump - удаляет базу данных постов и собирает новую по списку каналов, отбирает 50% самых просматриваемых постов за последние сутки. Период выборки: с 21:00 два дня назад до 21 прошлого дня.
  5. post - отправляет в целевой канал 3 ранее не отправленных поста вручную
  6. start - стартует процесс автоматического постинга в целевой канал в период 09:00-23:00
  7. stop - останавливает процесс автоматического постинга в целевой канал в период 09:00-23:00
  8. stats - статистика бота
  9. help - помощь

По умолчанию бот раз в сутки (08:00 GMT+3) удаляет базу данных постов и наполняет ее заново по списку каналов. В период 09:00-23:00 бот два раза в час отправляет в целевой канал по три поста.

Перешли боту видео, фото или альбом и он запостит их в целевой канал с текстовой ватермаркой
